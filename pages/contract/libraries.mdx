# Libraries

For security and efficiency concern, it's better to use utiltiy libraries to
write repeated code.

## [OpenZeppelin Contract](https://www.openzeppelin.com/contracts)

OZ provides a lot of useful contract such as `Ownable`, `ERC20` and `Pauseable`.

They also provide Proxy contract with upgradability pattern.

## [dapp.tools](https://dapp.tools/)

Famous for its ds-math, ds-test and ds-proxy.

## [Wyvern Protocol](https://wyvernprotocol.com/)

Opensea also uses its contract.

## [Gnosis Safe](https://gnosis-safe.io/)

Battle tested multisig contract for treasury and funds.

## [solidity-bytes-utils](https://github.com/GNSPS/solidity-bytes-utils)

Utility Solidity library composed of basic operations for tightly packed bytes
arrays.

## [solidity-stringutils](https://github.com/Arachnid/solidity-stringutils)

Basic string utilities for Solidity.

## [memmove](https://github.com/brockelmore/memmove)

A data structures library that does memory management for you.
